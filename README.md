# audio_IA

Proyecto de p5.js con la librería de sonido (p5.sound).

## Descripción

Este proyecto utiliza p5.js y la librería p5.sound para crear aplicaciones interactivas con audio. El proyecto incluye un ejemplo básico de un oscilador de sonido que puede ser controlado haciendo clic en el canvas.

## Características

- ✅ p5.js (versión 1.9.0)
- ✅ p5.sound library
- ✅ Ejemplo interactivo con oscilador de sonido
- ✅ Interfaz visual simple y responsiva

## Cómo usar

1. Abre el archivo `index.html` en un navegador web
2. Haz clic en el círculo del canvas para reproducir o pausar el sonido
3. El sonido es un tono sinusoidal de 440 Hz (nota A4)

## Estructura del proyecto

```
audio_IA/
├── index.html      # Archivo HTML principal con referencias a las librerías
├── sketch.js       # Código de p5.js con la lógica del sketch
├── style.css       # Estilos CSS para el proyecto
└── README.md       # Este archivo
```

## Tecnologías utilizadas

- **p5.js**: Framework de JavaScript para crear gráficos y experiencias interactivas
- **p5.sound**: Librería de extensión de p5.js para trabajar con audio
- **HTML5**: Estructura del proyecto
- **CSS3**: Estilos y diseño
